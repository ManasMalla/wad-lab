Getting started with HTML

HTML is the standard markup language for creating Web pages. It stands for Hyper Text Markup Language.

HTML describes the structure of a Web page. It uses markup tags to define each piece of content on the page.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
